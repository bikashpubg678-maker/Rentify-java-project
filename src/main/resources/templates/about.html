About Rentify

Rentify is a full-stack Car Rental Management System built as a 2nd-year Java project. It evolved from a basic console application into a production-ready web application demonstrating real-world software development skills including layered architecture, database persistence, PDF generation, and data visualization.

Tech Stack: Java 17, Spring Boot 3.2.5, Spring Data JPA, H2 Database, Thymeleaf, Chart.js, iText PDF, and integrated AI assistant powered by OpenRouter.

Features: Fleet management, date-based booking with double-booking prevention, PDF receipts, revenue charts, customer profiles, activity logging, loyalty discounts, and an AI chatbot that knows everything about the system.
